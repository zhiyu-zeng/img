> 💡 **AI 总结（key-points）**
>
> 通过保持bootloader版本一致并修改系统镜像配置文件，实现Pixel 6从Android 16安全降级到14。
> 
> - **降级风险：** 直接降级已升级的bootloader会导致黑砖，因此只降级系统，保持bootloader在16版本。
> - **前提检查：** 必须确保A、B分区的bootloader版本一致，可通过`fastboot getvar`命令切换检查，不一致则需先刷入统一版本。
> - **关键操作：** 刷入旧版系统前，需删除镜像包中`android-info.txt`文件里对bootloader和基带的版本要求行。
> - **刷机执行：** 在系统镜像文件夹内执行`fastboot flashall -w`，如遇报错需设置环境变量指向该路径。

由于手机自动更新导致升级到16，bootloader也升级到了16，无法退回旧版本的bootloader查阅相关资料发现降级会导致黑砖，所以不降级bootloader只降级系统。

![](https://cdn.jsdelivr.net/gh/zhiyu-zeng/img@main/img/2026/07/70a192f389ce9977.png)

#### 1.前提要保证a和b分区bootloader版本一致

版本不一致刷完有可能导致黑砖

```
# 重启进入bootloader模式
adb reboot bootloader

# 查看当前分区版本
fastboot getvar current-slot
fastboot getvar version-bootloader

#切换分区b （如果切换回去查看完，要记得切换回来）
fastboot --set-active=b
```

可以看到当前是a分区版本是 16.4-14548185

![](https://cdn.jsdelivr.net/gh/zhiyu-zeng/img@main/img/2026/07/5de1456d800718e6.png)

切换到b分区，版本也是一样的（查看后，注意要切换回去a分区）

![](https://cdn.jsdelivr.net/gh/zhiyu-zeng/img@main/img/2026/07/bd1dd206949ad29f.png)

#### 2.如果bootloader版本不一样或者不确定，下载当前系统版本镜像，将a和b分区都刷一遍

```
# 将当前分区刷入bootloader
fastboot flash bootloader bootloader-oriole-slider-16.4-14548185.img
# 将另外一个分区刷入bootloader
fastboot --slot=other flash bootloader bootloader-oriole-slider-16.4-14548185.img
```

下载当前系统镜像包，找到bootloader镜像

![](https://cdn.jsdelivr.net/gh/zhiyu-zeng/img@main/img/2026/07/b3e1cc7e76c859c8.png)

对2个分区都刷一遍

![](https://cdn.jsdelivr.net/gh/zhiyu-zeng/img@main/img/2026/07/c6e29cd1a6c1dc59.png)

#### 3.刷入14系统镜像

将包解压

![](https://cdn.jsdelivr.net/gh/zhiyu-zeng/img@main/img/2026/07/053b12a1afea478d.png)

打开解压后的文件夹，找到android-info.txt，将

require version-baseband=g5123b-120690-230802-B-10596884

require version-bootloader=slider-1.3-10674934

都删掉，再保存。

![](https://cdn.jsdelivr.net/gh/zhiyu-zeng/img@main/img/2026/07/12ec4a7c532b3578.png)

在解压的文件夹下执行刷机命令

```
fastboot flashall -w
```

会报错，设置系统环境到当前文件夹路径下即可。

![](https://cdn.jsdelivr.net/gh/zhiyu-zeng/img@main/img/2026/07/ead8142ed1b01a24.png)

![](https://cdn.jsdelivr.net/gh/zhiyu-zeng/img@main/img/2026/07/39d1423dbd98ef0e.png)

设置完成后，要重新cmd窗口打开，再执行一次fastboot flashall -w，刷完开机后，就降级成功了，再进入开发者模式把自动更新关了，以免下次自动更新。

![](https://cdn.jsdelivr.net/gh/zhiyu-zeng/img@main/img/2026/07/df0589683aa7b29b.png)
