## 安卓完美云手机方案定制实战

一款个人认为目前最完美的安卓云手机方案。

## 安卓云手机目标

-   支持国内外主流安卓机型
    
-   支持安卓15/16/17
    
-   支持macOS系统运行部署
    
-   支持全系列2025年后arm64电脑与开发板运行
    
-   全系统功能完整
    
-   真实硬件检测信息
    
-   真实可运行国内外主流软件
    

## 运行要求

-   macOS目前支持Apple Silicon Macmini M4 Pro。
    
-   Linux只支持arm64，启动前必须确认 `/dev/kvm` 可读写。
    
-   Pixel10 / Pixel10Pro 当前推荐宿主是 CIX P1 的 Orion O6N 开发板；该路线已经测试成功完美运行。其它 CIX P1 相关 Linux arm64/KVM 方案也可以跑。
    
-   `just run` 必须使用 `output/qemu-cdh/qemu-system-aarch64` ，不使用系统 `qemu-system-aarch64` 。
    
-   启动内存不能低于4096MB，宿主可探测物理内存低于4GB时直接失败。
    
-   系统可用空间不能低于50GB。

## FAQ

问：本方案是基于redroid还是lxc还是AVD还是CVD？

答：这些方案都不是，本课程采用另一种全新更完美的技术方案。

问：课程中讲的方案支持哪些安卓设备定制？

答：理论现有的安卓设备都支持。

问：课程中会讲解哪些安卓型号的定制？

答：官方Pixel与国产某米某加与三星等主流机型。

问：小白可以学习这个课程吗？

答：小白不适合。云机系统定制需要有一定的开发与逆向工程基础。在学习全系列课程内容前，最少需要先看安卓系统定制与安卓安全开发与逆向分析全套课程。

问：学这个课程能获得什么？

答：可以得到：

```
全套云手机无保留实现思路以及方案的特点实现方式与多开部署方法。

节省时间Token与少踩坑，节省时间。课程中会把所有的踩坑记录与学员全部分享。

多款已经实现好的云手机镜镜直接无限制使用。

多款云手机特点与Bug处理方法与定制全过程。

全部云手机过程的profile文档与核心知识点的代码。

云手机管理器与镜像长达半年功能新增与Bug修正支持。
```

问：学这个课程不能获得什么？

答：以下场景：

```
支持国产系统自主研发，虽然讲解了国产某些型号的全部过程，但由于我国法律法规，不直接提供部分国产定制好的镜像。国产云机镜像产物，学员在学习完本系列课程后可完全自主100%定制实现。

不提供任何华为设备的云手机化。虽然本课程方案支持。

不提供全部（课程中讲的全部提供）的云手机项目的代码与文档。完整项目的文档代码与更新服务可另外商定。

不提供过脸部验证、视频通话、语音通话、GPS定位等高风险功能支持。本项目云手机如果引入这些功能，也不会是主要的卖点。如果学员有这方面需求，可以在课程方案基础上自行定制（方案原理上完全支持），或者寻求其它方案。

不提供云手机定制服务。学员提出的需求如果是云手机自身功能限制需求，可考虑完善后提供给学员二进制使用。
```
