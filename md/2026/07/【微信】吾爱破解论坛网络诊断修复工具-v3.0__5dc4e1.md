## 吾爱破解论坛网络诊断修复工具

面向吾爱破解论坛（www.52pojie.cn）用户的 Windows 网络诊断修复工具，一键检测并修复论坛无法访问、访问速度缓慢、DNS 劫持等常见网络问题。

> **系统要求** ：仅支持 Windows 10 及以上版本的系统。

### 功能特性

-   **一键修复**
    
    ：点击「一键修复」自动深度检测网络环境
    
-   **访问异常排查**
    
    ：解决论坛无法访问的问题
    
-   **速度优化**
    
    ：修复网络访问速度缓慢的问题
    
-   **DNS 劫持检测**
    
    ：检测并排查 DNS 劫持
    
-   **访客模式排查**
    
    ：验证码 / 登录 / 注册 / 发帖异常时，可切换浏览器访客模式排查
    
-   **日志反馈**
    
    ：遇到问题可将诊断日志发送至 service@52pojie.cn，或前往论坛站务区发帖求助
